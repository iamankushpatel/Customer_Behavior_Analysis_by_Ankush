# Customer_Behavior_Analysis_by_Ankush
data analytics project showcasing customer behavior analysis using python, sql and power Bi.

This project demonstrates a complete data analytics workflow, transforming raw data into actionable business insights. The process integrates multiple industry-standard tools to handle data cleaning, relational database querying, and interactive visualization.

Project Stages
Data Preparation: Used Python (Pandas) to clean messy raw data, handle missing values, and perform Exploratory Data Analysis (EDA) to identify initial patterns.

Database Management: Processed the cleaned data using SQL to calculate key performance indicators (KPIs), such as total revenue, top-selling products, and regional sales performance.

Visualization: Developed an interactive Power BI dashboard to present complex datasets through intuitive charts, allowing for real-time filtering and trend analysis.

Communication: Consolidated findings into a formal project report and a presentation deck created via Gamma to translate technical data into business recommendations.

Key Learning & Growth
A primary focus of this project was analyzing revenue distribution across different demographics. While the current version focuses on total revenue by age, it established a foundation for more advanced customer segmentation and demographic grouping in future iterations.

Core Technical Stack
Languages: Python, SQL

Database: PostgreSQL / MySQL / SQL Server

BI Tool: Power BI

Reporting: Gamma, Microsoft Word/PPT
